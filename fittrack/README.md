# FitTrack 💪

Gym Members ke liye Daily Workout Tracker App

## Features
- ⚡ 7 Muscle Groups (Chest, Back, Shoulders, Arms, Legs, Core, Full Body)
- 📊 3 Fitness Levels (Beginner, Intermediate, Advanced)
- ⏱️ Rest Timer with beep sound
- 🏆 Personal Record (PR) Tracker
- 📅 Weekly workout calendar
- 💾 Auto-save history & streak

## Setup

```bash
npm install
npm start
```

## Build for Production
```bash
npm run build
```
